# NutriScan

👉 **Visit our website:** [NutriScan Website](https://nutriscaner.pythonanywhere.com/)

NutriScan is a nutrition-focused web application designed to help users analyze and track their dietary intake. By scanning food products or manually entering nutritional information, NutriScan provides insights into the nutritional content of meals and helps users make informed dietary decisions.

## Features

- **Food Scanning**: Scan food items and retrieve nutritional information.
- **Manual Entry**: Enter nutritional information manually for custom meals or items.
- **Nutritional Analysis**: Receive detailed breakdowns of calories, macronutrients, vitamins, and minerals.
- **User Profiles**: Create and manage user profiles to track your nutrition over time.
- **Healthy Eating Tips**: Get personalized recommendations for a balanced diet.
